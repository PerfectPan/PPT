入门题

巴什博弈

Alice 和 Bob 在玩游戏，有 n 颗石子，两人轮流取走石子，每次可以取 1 - m 颗石子。Alice 先手，取走最后一颗石子的人获胜。

问：Alice 先手必胜么？

公平组合游戏定义

若一个游戏满足：

  • 双人游戏，两名玩家交替行动，两个人都绝顶聪明，每一次都是选择最优的策略行动
  • 合法的行动是一个有限的状态集合，意味着游戏可以在有限步结束
  • 在游戏进程的任意时刻，可以执行的合法行动与轮到的哪名玩家无关
  • 不能行动的玩家判负

Q：围棋/象棋算公平组合游戏么？

必胜必败态

博弈图

如果我们将每一步走完的游戏局面视为一个节点，再从每个节点向它的后继节点连边，我们就可以得到一张博弈状态图，且这张图是有向无环图。

必胜必败态定理

定义必胜状态为先手必胜的状态，必败状态为先手必败的状态：

  1. 没有后继状态的状态是必败状态。
  2. 一个状态是必胜状态当且仅当存在至少一个必败状态为它的后继状态。
  3. 一个状态是必败状态当且仅当它的所有后继状态均为必胜状态。

还是以刚才的入门题为例，我们对博弈状态图进行推导：

必胜必败态定理

因此，对于很多公平组合游戏，我们能表示出来当前游戏的状态，就可以利用必胜必败态定理在博弈图上拓扑逆序推导出每个节点是必胜态还是必败态。

假设 N 为状态总数，M 为边数，那么时间复杂度即为 O(N+M)。

例题散讲

Ferguson Game

Alice 和 Bob 两个人在玩游戏。当前有两个盒子，一开始其中一个有 m 颗糖，另一个有 n 颗糖。每次移动是将一个盒子清空，而把另一个盒子的一些糖拿到被清空的盒子中，使得两个盒子至少各有一颗糖，最后不能移动的人判负，问 Alice 是否必胜？

数据范围：0 < n <= 1e9, 0 < m <= 1e9

Hint：不妨用必胜必败态定理打个表试一下？

A Plate Game

Alice 和 Bob 两个人在玩游戏。当前有一张指定大小(w x h)的矩形桌子和一个半径为 r 的圆盘。两人轮流往桌子上放圆盘，要求放置的时候圆盘不能重叠，不能一部分摆出桌子外，不能摆放的人就失败了，Alice 先手，问 Alice 是否必胜？

数据范围：1 <= w, h, r <= 100

总结

Take-away message

  • 公平组合游戏本质上可以转化为一个有向无环图，利用必胜必败态定理依据逆拓扑序推出所有节点的状态
  • 构造游戏的对称性是解决公平组合游戏的一个思考方向
